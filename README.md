# GitHub Proyecto Final

Repositorio de práctica profesional para dominar Git y GitHub.

## Objetivo
Aplicar flujo profesional con ramas, commits, pull requests, GitHub Actions, branch protection y releases.

## Tecnologías
- Git
- GitHub
- GitHub Actions
- VS Code

## Estructura
- `docs/`: documentación de prácticas y flujo de trabajo
- `.github/workflows/`: automatizaciones
- `.github/pull_request_template.md`: plantilla de PR
- `.github/CODEOWNERS`: responsables del repo

## Flujo de trabajo
1. Crear rama
2. Hacer cambios
3. Revisar diff
4. Hacer commit
5. Hacer push
6. Abrir pull request
7. Esperar CI y revisión
8. Hacer merge
